2. DESIGN LANGUAGE & AESTHETIC
================================================================================

  Overall Theme: "Newsprint / Retro Terminal" Hybrid

  Color Palette (CSS Variables):
  - Background Primary: #F9F9F7 (warm off-white, paper-like)
  - Foreground Primary: #111111 (near-black)
  - Muted: #E5E5E0 (light gray)
  - Accent: #CC0000 (red accent)

  Typography (CSS Variables):
  - Serif (Headings): "Playfair Display", serif
  - Body: "Lora", Georgia, serif
  - Sans: "Inter", sans-serif
  - Mono (Terminal/Code): "JetBrains Mono", monospace

  Key Design Elements:
  - 2px solid borders on all sections/elements
  - Hard shadows on hover: 4px offset, solid black
  - Newsprint texture: SVG dot pattern overlay (subtle)
  - Terminal-style typing animation for hero text
  - Grid-based layouts (12-column system in some areas)
  - Uppercase, wide letter-spacing for labels (0.18em-0.2em)

  Component Patterns:
  - Sections wrapped in bordered containers
  - Hard shadow hover effect with translate (-2px, -2px)
  - Tables for data display
  - Card-based project layouts

4. CODE TECHNICALITIES
================================================================================

  Entry Point Flow:
  1. index.html loads /src/main.tsx
  2. main.tsx renders App component in StrictMode
  3. App.tsx contains entire portfolio as single-page application

  Key React Patterns:
  - useState: For typing animation state (typedHero)
  - useEffect: For typewriter effect on mount
  - Component structure: Single App component with multiple sections
  - No external component library (pure React + Tailwind)

  Tailwind Configuration (v4):
  - Uses @import "tailwindcss" (v4 syntax, not @tailwind directives)
  - Custom @theme block for CSS variables
  - @layer base for global styles
  - @layer utilities for custom utility classes

  Custom Utilities:
  - .newsprint-texture: Adds paper-like crosshatch overlay
  - .hard-shadow-hover: Box-shadow animation on hover

  Responsive Design:
  - Mobile-first with md: and lg: breakpoints
  - max-w-6xl for main content width
  - Grid layouts: 1 col mobile, 2-3 col on larger screens


5. SCRIPTS (npm run ...)
